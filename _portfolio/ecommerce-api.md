# E-Commerce Platform API

A highly scalable backend API for a modern e-commerce platform. Features include:

- User authentication and authorization (JWT).
- Product catalog management with faceted search.
- Shopping cart and order processing.
- Payment gateway integration.
- Caching with Redis for high-traffic endpoints.

This project demonstrates my ability to design and implement robust, high-performance web services.
